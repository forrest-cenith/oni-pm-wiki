# ONI PM Dashboard

## Overview
The ONI PM Dashboard is a super-user tool on the marketplace that lets you manage PM data for all challenges. It streamlines the Challenge Closeout process by letting you search challenges, view all submissions and their files, download them, and generate Tech Eval documents directly.

## Getting Access

You need a **super user** account to access the ONI PM tab. Regular marketplace accounts won't see it.

### 1. Create a New Marketplace Account
- Go to the marketplace and create a new user with your real email using the `+super` alias
  - Example: `forrest.parker+super@onenationinnovation.org`
  - Emails will still route to `forrest.parker@onenationinnovation.org`, but the marketplace treats it as a separate user
- **Do NOT affiliate this account with an org** — it needs to stay unaffiliated

### 2. Request Super User Privileges
- Message Forrest and ask him to upgrade this account to super user
- Once granted, you'll see the **ONI PM** tab in the left sidebar

## Using the Dashboard

Once you have access:

1. **Search for a challenge** using the search bar at the top
2. **Click on the challenge** to open it — you'll see all submissions listed with their files
3. **Select and download** submission files as needed for the closeout package
4. Use the **Download Technical Evaluation Report** button to generate the tech eval document

![ONI PM Dashboard — challenge view with submissions and action buttons](/img/oni-pm-dashboard.png)

## How This Helps with Challenge Closeout
The ONI PM Dashboard replaces several manual steps in the [Challenge Closeout Package](/docs/marketplace-workflows/02-challenge-closeout-package) workflow:
- No need to individually download submissions from the public challenge page
- Bulk select and download submission files
- Generate the Tech Eval document directly instead of filling it out manually
