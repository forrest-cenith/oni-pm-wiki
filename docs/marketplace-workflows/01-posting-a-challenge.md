# Posting a Challenge on the Marketplace

## Overview
Step-by-step guide for creating and publishing a challenge on the Colosseum Marketplace after the government customer has filled in their challenge template.

**Related pipeline stage:** [Challenge Live](/docs/challenge-pipeline/03-challenge-live)

## Steps

1. **Send the government customer the templates** (see [OTA Intro Call](/docs/challenge-pipeline/01-ota-intro-call))
   - LOE + Challenge Template (combined doc): [Google Doc](https://docs.google.com/document/d/1ckzEEqWsXD6J7rqkfsc_lQKxrRHFFOa1/edit?usp=drive_link&ouid=116482897251814755876&rtpof=true&sd=true)

2. **Go to the marketplace and create a challenge**

3. **Set the name**
   - Format: `[[Gov Org Name]] - [[Challenge Name]]`

4. **Set the points of contact**
   - scott.helmore@onenationinnovation.org
   - stephen.reynolds@onenationinnovation.org
   - amy.asberry@onenationinnovation.org
   - forrest.parker@onenationinnovation.org

5. **Set the dates**
   - Start Date: Date of posting
   - End Date: ~3 weeks later, Wednesday at noon (optimizes support coverage)

6. **Set the challenge fields from the completed template**
   - Description = GovFilledChallengeTemplate.description
   - Overall Objective = GovFilledChallengeTemplate.overallObjective
   - Problem Statement = GovFilledChallengeTemplate.problemStatement
   - Desired Solution = GovFilledChallengeTemplate.desiredSolution

7. **Attach supporting documents**
   - LOE / Challenge Detail document
   - CUI Notice
   - ONI OTA Agreement

## Tips
- End date on a Wednesday at noon optimizes ONI support coverage for last-minute submissions
- Always double-check that all four challenge fields match the government's completed template
- Attach all mandatory supporting documents before publishing
