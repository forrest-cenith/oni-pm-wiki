# Invoice Intake Checklist

Use for every vendor invoice received.

## Receipt
- [ ] Date received logged
- [ ] Invoice number recorded
- [ ] Billing period / milestone identified

## Validation Against Contract
- [ ] Invoice references correct contract number
- [ ] Invoice references correct modification (if applicable)
- [ ] Billed CLINs match contract CLIN structure
- [ ] Amounts align with contract/agreement values
- [ ] No unapproved scope is billed
- [ ] Period of performance is within contract POP

## Supporting Documentation
- [ ] Backup documentation provided
- [ ] Deliverable acceptance records available
- [ ] Milestone completion evidence on file
- [ ] Math is correct (line items sum to total)

## CLIN Mapping
- [ ] Each vendor line item mapped to ONI CLIN
- [ ] ONI invoice totals match vendor invoice totals
- [ ] CLIN descriptions are accurate

## Discrepancy Resolution (if needed)
- [ ] Discrepancies documented
- [ ] Vendor notified with specific items needing correction
- [ ] Corrected invoice received and re-validated

## Ready for ONI Invoice Generation
- [ ] All validation checks passed
- [ ] ONI invoice generated with correct CLIN mapping
- [ ] Required attachments included
- [ ] Internal approval obtained
