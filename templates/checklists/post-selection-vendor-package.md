# Post-Selection Vendor Package Checklist

Send to vendor immediately after selection notification.

## Required Documents
- [ ] Updated proposal aligned to government SOW/LOEs
- [ ] Revised SOW (if vendor-authored)
- [ ] W-9 (tax identification)
- [ ] Reps and Certs (vendor certifications)
- [ ] Confirmed ROM and milestone/payment schedule alignment
- [ ] Any additional required administrative forms

## PM Verification Before Sending to ACC-RI
- [ ] Vendor proposal aligns with government SOW scope
- [ ] Deliverables in proposal match SOW deliverables
- [ ] Milestones and timeline are consistent across documents
- [ ] ROM/cost figures match between proposal and SOW
- [ ] All administrative forms are complete and signed
- [ ] Terminology is consistent across all documents
- [ ] Package assembled as single authoritative version

## Submission to ACC-RI
- [ ] One clean package assembled (not scattered fragments)
- [ ] Submission email sent with all attachments
- [ ] Submission date logged in tracker
- [ ] Copy saved of exactly what was submitted
